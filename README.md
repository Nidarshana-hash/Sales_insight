# Customer Shopping Behavior Analysis

## Overview

This project demonstrates an end-to-end data analytics workflow using Python, PostgreSQL, and Power BI. The goal is to analyze customer shopping behavior by performing data cleaning, exploratory data analysis (EDA), SQL-based analysis, and creating an interactive dashboard for business insights.

---

## Dataset

- **Dataset:** Customer Shopping Behavior Dataset
- **Format:** CSV
- **Source:** Public dataset
- **Records:** Customer purchase transactions including demographics, product details, payment methods, ratings, and purchase amounts.

---

## Tools & Technologies

- **Python**
  - Pandas
- **PostgreSQL**
- **SQL**
- **Power BI**
- **Jupyter Notebook**

---

## Project Workflow

### 1. Load Dataset
- Import the CSV file using Pandas.
- Inspect the dataset structure and data types.

### 2. Exploratory Data Analysis (EDA)
- Understand the distribution of data.
- Analyze customer demographics.
- Explore purchase trends.
- Visualize important patterns using charts.

### 3. Data Cleaning
- Handle missing values.
- Remove duplicate records.
- Check data types.
- Prepare the dataset for analysis.

### 4. SQL Analysis
- Import the cleaned dataset into PostgreSQL.
- Execute SQL queries to answer business questions.
- Perform filtering, grouping, aggregation, and sorting.

### 5. Power BI Dashboard
Create an interactive dashboard to visualize:
- Total Sales
- Average Purchase Amount
- Customer Gender Distribution
- Product Category Analysis
- Payment Method Distribution
- Shopping Trends
- Customer Ratings

---

## Dashboard

The Power BI dashboard provides an interactive view of customer shopping behavior with filters and visualizations that help identify purchasing trends and customer preferences.

---

## Results

Key insights include:
- Customer purchasing patterns.
- Popular product categories.
- Preferred payment methods.
- Gender-wise shopping trends.
- Overall sales performance.
- Customer rating analysis.
